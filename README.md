# Sneaky
Snake Game 2D using OpenGl.

---

## Compiling & Running
### 1. Install requirments & build
  ```bash
    make build
  ```
### 2. Run
  ```bash
    make run
  ```
### 3. Clean
  ```bash
    make clean
  ```
