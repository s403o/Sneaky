# Sneaky
Snake Game 2D using OpenGl.

---

## Compiling & Running
### 1. Install OpenGl librares (gl.h && glut.h) & Zenity.
  ```bash
  sudo apt install freeglut3-dev zenity
  ```
### 2. Compile & Run.
  ```bash
  g++ main.cpp -o sneaky -lGL -lGLU -lglut && ./sneaky
  ```
